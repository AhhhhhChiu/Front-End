# 响应式原理

在vue3.x中是通过Proxy来实现响应式数据的

```js
const obj = { num: 1 };
const state = new Proxy(obj, {
    get(obj, key) {
        console.log('get', obj[key]);
        return obj[key];
    },
    set(obj, key, value) {
      console.log('set', obj[key], value);
      obj[key] = value;
      // 注：需要返回true表示当前操作已经成功
      return true;
    }
});

state.num = 2; // set 1 2
const num2 = state.num; // get 2
```
